# FinSight AI — Intelligent Financial Platform

FinSight AI is a production-ready, full-stack financial intelligence application designed to help users track expenses, analyze spending behaviors, scan invoices, monitor investments in real-time, and receive AI-driven forecasts and budget recommendations.

## Live Deployment Links
- **Frontend Application**: [https://finsight-ai-expense-and-investment-tracker.vercel.app/](https://finsight-ai-expense-and-investment-tracker.vercel.app/)
- **Backend API Service**: [https://finsight-ai-1-fp07.onrender.com](https://finsight-ai-1-fp07.onrender.com)

---

## Key Features

- **Strict Identity Verification**: Robust credential signup enforcing enterprise password policies (length, uppercase, lowercase, numbers, and special characters) using secure Bcrypt hashing.
- **AI-Powered Bill Scanner**: Upload or capture invoices directly. An integrated Gemini API multimodal model extracts transaction details (total amount, merchant name, date, and category) to pre-fill logs.
- **Smart Expense Categorization**: Uses Gemini to auto-categorize typed expense titles on-the-fly.
- **Live Investment Portfolio**:
  - Log stocks, crypto assets, or commodities (e.g. `AAPL`, `BTC-USD`, `GC=F`).
  - Automatically fetches live asset valuation quotes and net gain/loss status using the Yahoo Finance API.
- **Python Data Analytics Layer**:
  - **Expense Forecasting**: Fits a Scikit-learn Linear Regression model to historical transactions to project the next 30 days of expenditures.
  - **Financial Health Scoring**: Computes a dynamic health score (0-100) using Pandas and NumPy, analyzing savings ratios, discretionary vs. essential budgets, and volatility coefficients.
- **FinSight AI Recommendations**: Returns customized, highly specific saving strategies generated by the Gemini model.
- **Microsoft Power BI Integration**: Renders public Power BI embed frames dynamically based on URLs stored in user preferences. Includes native interactive Recharts mockup dashboards when no Power BI URL is provided.
- **SaaS-Grade Responsive Theme**: A clean light UI (pure whites, soft slates, deep slate fonts, and sharp indigo accents) styled with Tailwind CSS, custom scrollbars, and Framer Motion micro-animations.

## Configuration & Installation

### 1. Backend Setup
1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure the environment (`backend/.env`):
   Create a `.env` file and specify the variables:
   ```env
   PORT=5000
   DATABASE_URL=postgres://user:password@localhost:5432/finsight_db
   JWT_SECRET=your_jwt_secret_key
   GEMINI_API_KEY=your_gemini_api_key_here
   ```
   *Note: If no `DATABASE_URL` is configured or connection fails, the system automatically spins up a local SQLite database (`database.sqlite`). If no `GEMINI_API_KEY` is provided, the system falls back to simulated AI analysis.*
4. Start the server:
   ```bash
   npm run dev
   ```

### 2. Frontend Setup
1. Open a new terminal and navigate to `frontend`:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the client:
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser.

## Strict Password Strength Rules
Passwords must strictly satisfy:
- Minimum **8 characters**.
- At least one **uppercase letter** (`A-Z`).
- At least one **lowercase letter** (`a-z`).
- At least one **digit** (`0-9`).
- At least one **special character** (e.g., `!@#$%^&*`).
